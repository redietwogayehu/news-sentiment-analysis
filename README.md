# news-sentiment-analysis & Stock Market Prediction
Financial news sentiment analysis and stock correlation project (Week 1 AI Mastery)


# Project Objective

To build a data pipeline that:
- Extracts sentiment from financial news headlines
- Analyzes stock price movements
- Identifies relationships between sentiment and market behavior
- Applies technical indicators for quantitative analysis

---

## Task 1 — Exploratory Data Analysis (EDA)

In Task 1, we analyzed financial news data:

### Key Work Done:
- Loaded and cleaned news dataset
- Converted publication dates to datetime
- Analyzed headline lengths
- Identified top publishers
- Visualized daily news volume trends
- Extracted patterns in financial news activity
